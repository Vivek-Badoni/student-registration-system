# PHP CRUD Application

A simple and responsive CRUD (Create, Read, Update, Delete) application built using PHP, MySQL, Bootstrap, jQuery, DataTables, and SweetAlert. This project allows users to manage records with form validation, file upload, and a clean user interface.

---

## 🚀 Features

- Add new records
- View all records in a DataTable
- Update existing records
- Delete records with confirmation alert
- Upload files/images
- Client-side validation using jQuery Validation
- Responsive UI with Bootstrap
- Search, Sorting, and Pagination using DataTables
- SweetAlert confirmation and success messages

## 💻 Installation

1. Download or Clone the repository

```bash
git clone https://github.com/YourUsername/PHP-CRUD.git
```

2. Move the project folder to

```
xampp/htdocs/
```

3. Start

- Apache
- MySQL

from XAMPP Control Panel.

4. Open phpMyAdmin.

5. Create a database

```
crud_db
```

6. Import the SQL file.

7. Open your browser and run

```
http://localhost/PHP-CRUD/
```
